# ChessGame - 国际象棋游戏

一个使用纯 HTML、CSS 和 JavaScript 开发的交互式国际象棋游戏。

## 功能特点

- ✅ 完整的国际象棋规则实现
- ✅ 直观的图形化棋盘界面
- ✅ 拖拽选择和移动棋子
- ✅ 有效移动位置高亮显示
- ✅ 将军、将死、和棋检测
- ✅ 移动历史记录和撤销功能
- ✅ 被吃棋子显示
- ✅ 响应式设计，支持移动设备
- ✅ 键盘快捷键支持

## 游戏规则

### 基本操作
1. 点击要移动的棋子进行选择
2. 绿色高亮显示可移动的位置
3. 红色高亮显示可以吃子的位置
4. 点击目标位置完成移动

### 特殊规则
- **将军检测**：王被攻击时会有红色闪烁提示
- **将死判定**：王被将军且无法脱离危险时游戏结束
- **和棋判定**：当前玩家无合法移动但未被将军时为和棋
- **兵的升变**：兵走到对方底线自动升变为后

### 快捷键
- `Esc` - 取消当前选择
- `Ctrl+N` - 新游戏
- `Ctrl+Z` - 撤销上一步
- `Ctrl+R` - 重置棋盘

## 技术实现

### 棋子系统 (`pieces.js`)
- `ChessPiece` 类：定义棋子的属性和移动规则
- 支持所有棋子类型：王、后、车、象、马、兵
- 实现各棋子的合法移动判断

### 棋盘系统 (`board.js`)
- `ChessBoard` 类：管理棋盘状态和游戏规则
- 移动验证和游戏状态检测
- 历史记录管理和撤销功能

### 游戏控制 (`game.js`)
- `ChessGame` 类：管理用户界面和交互
- 棋盘渲染和事件处理
- 游戏状态显示更新

### 主程序 (`main.js`)
- 游戏初始化和事件绑定
- 键盘快捷键处理
- 调试功能和统计系统

## 开发环境

### 运行要求
- 现代浏览器（支持 ES6+）
- 本地服务器（推荐）或直接打开 HTML 文件

### 本地开发
1. 克隆或下载项目文件
2. 使用 VS Code Live Server 或其他本地服务器
3. 打开 `index.html` 开始游戏

### 在 VS Code 中运行
1. 安装 Live Server 扩展
2. 右键点击 `index.html`
3. 选择 "Open with Live Server"

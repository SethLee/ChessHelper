# ChessGame - 中式象棋变体游戏

一个9×8网格的中国象棋变体游戏，集成AI智能推荐系统，使用纯 HTML、CSS 和 JavaScript 开发。

## 🎮 功能特点

- ✨ **独特的9×8棋盘设计** - 基于交叉点的中式布局
- 🤖 **AI智能推荐** - 红车专属最佳落点分析
- 🎯 **实时安全检测** - 智能威胁评估和路径分析
- 🎨 **直观的可视化反馈** - 金色星标标识最佳位置
- 📱 **完全响应式设计** - 支持桌面和移动设备
- 🔧 **便捷的棋子管理** - 点击放置、删除黑子功能
- 🎪 **流畅的用户体验** - 快速弹窗、即时响应

## 🎲 游戏规则

### 基本操作
- **放置棋子**：点击空白位置，选择要放置的棋子类型
- **移动棋子**：点击棋子选择，再点击目标位置移动
- **删除棋子**：点击黑色棋子显示删除按钮

### 视觉指示
- 🟢 **绿色高亮**：可移动的安全位置
- ⭐ **金色星标**：AI推荐的最佳落点

### 棋子移动规则
- **车**：直线移动，可跨越多格
- **马**：日字形移动，不能被阻挡
- **象**：田字形移动，2格对角线
- **士**：对角线移动1格
- **将**：横竖移动1格
- **炮**：直线移动，需跳跃吃子
- **卒**：横竖移动1格

## 🔧 技术实现

### 棋子系统 (`pieces.js`)
- `ChessPiece` 类：中国象棋棋子定义
- 支持7种棋子：车、马、象、士、将、炮、卒
- 实现各棋子的移动规则和路径验证

### 棋盘系统 (`board.js`)
- `ChessBoard` 类：9×8交叉点棋盘管理
- 棋子放置、移动验证和状态管理
- 攻击检测和安全性分析

### AI系统 (`game.js`)
- 智能最佳移动计算算法
- 多因素评分系统（安全性、威胁值、位置控制）
- 红车专属路径优化和威胁分析

### 用户界面 (`game.js`)
- 响应式SVG网格渲染
- 模态弹窗棋子选择系统
- 实时视觉反馈和动画效果

### 主程序 (`main.js`)
- 游戏初始化和事件管理
- 响应式布局适配
- 调试和统计功能

## 🚀 快速开始

### 在线演示
```bash
# 使用Python启动本地服务器
python -m http.server 8000
# 访问 http://localhost:8000
```

### 本地运行
1. 克隆项目：`git clone https://github.com/SethLee/ChessGame.git`
2. 进入目录：`cd ChessGame`  
3. 启动服务器：`python -m http.server 8000`
4. 浏览器打开：`http://localhost:8000`

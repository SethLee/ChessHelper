# ChessHelper - 仙奕破阵辅助工具

一个专为仙奕破阵设计的红车走法分析工具，基于9×8网格棋盘，集成AI智能推荐系统，使用纯 HTML、CSS 和 JavaScript 开发。

## 🎮 功能特点

- ✨ **独特的9×8棋盘设计** - 基于交叉点的中式布局
- 🤖 **AI智能推荐** - 红车专属最佳落点分析算法
- 🎯 **实时安全检测** - 智能威胁评估和路径优化
- 📋 **预设功能** - 可预设未来棋子和回合计数系统
- 🎨 **直观的可视化反馈** - 金色星标标识AI推荐位置
- 📱 **完全响应式设计** - 支持桌面和移动设备
- 🔧 **便捷的棋子管理** - 点击放置、删除黑子功能
- 🎦 **流畅的用户体验** - 快速弹窗、即时响应

## 🎲 使用说明

### 基本操作
- **预设功能**：点击加减号调整回合数，点击圆形虚线框预设新增棋子
- **分析走法**：点击红车查看AI推荐的最佳移动位置
- **布局调整**：点击空白位置或黑子可手动调整布局
- **回合系统**：移动黑子会减少回合数，归零时需拖拽放置棋子

### 视觉指示
- 🟢 **绿色高亮**：红车可移动的安全位置
- 🔵 **蓝色高亮**：红车可吃子的位置
- ⭐ **金色星标**：AI推荐的最佳落点
- 🔴 **红色警告**：危险的移动位置

### 棋子移动规则
- **车**：直线移动，可跨越多格
- **马**：日字形移动，不能被阻挡
- **象**：田字形移动，2格对角线
- **士**：对角线移动1格
- **将**：横竖移动1格
- **炮**：直线移动，需跳跃吃子
- **卒**：横竖移动1格

## 🔧 技术实现

### 棋子系统 (`pieces.js`)
- `ChessPiece` 类：仙奕破阵棋子定义
- 支持7种棋子：车、马、象、士、将、炮、卒
- 实现各棋子的移动规则和路径验证

### 棋盘系统 (`board.js`)
- `ChessBoard` 类：9×8交叉点棋盘管理
- 棋子放置、移动验证和状态管理
- 攻击检测和安全性分析

### AI系统 (`game.js`)
- 智能最佳移动计算算法
- 多因素评分系统（安全性、威胁值、位置控制）
- 红车专属路径优化和威胁分析
- 预设棋子系统和回合计数功能

### 用户界面 (`game.js`)
- 响应式SVG网格渲染
- 模态弹窗棋子选择系统
- 实时视觉反馈和动画效果
- 预设席位拖拽系统和警告提示

### 主程序 (`main.js`)
- 分析初始化和事件管理
- 响应式布局适配
- 调试和统计功能

## 🚀 快速开始

### 在线演示
```bash
# 使用Python启动本地服务器
python -m http.server 8000
# 访问 http://localhost:8000
```

### 本地运行
1. 克隆项目：`git clone https://github.com/SethLee/ChessHelper.git`
2. 进入目录：`cd ChessHelper`  
3. 启动服务器：`python -m http.server 8000`
4. 浏览器打开：`http://localhost:8000`
